# ListingPro AI 🏠🇰🇪

An AI-powered advertisement generator designed specifically for real estate agents in the Kenyan market.

## ✨ Features
- **Instant Ad Generation:** Converts property specs into professional, high-converting descriptions.
- **Local Optimization:** Tailors content for Nairobi, Nakuru, and other local markets.
- **Smart Google Maps Links:** Automatically generates a map link based on location input.
- **Tone Selection:** Choose between Professional, Persuasive, or Casual styles.
- **Mobile First:** Fully responsive design for agents on the go.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6)
- **AI Engine:** Llama 3.3 via Groq API
- **Deployment:** Vercel

## 🚀 How it Works
1. Enter property details (Price, Location, Bedrooms, etc.).
2. Select the desired tone for the ad.
3. The AI analyzes the market specs and generates a formatted listing ready for WhatsApp, Jiji, or Facebook.

## 🔒 Security
This project uses **Environment Variables** to manage API keys, ensuring that sensitive credentials are never exposed in the source code.
